## 用例1（患者A）——典型HCC，完整结构化【考点：准确率+覆盖度】

**这条展示**：从门诊+病程+MRI+病理多份文书中完整提取，病理确诊HCC，免疫组化解析，影像"快进快出+假包膜"识别。

📋 **输入文本**：
```
请对以下肝病病历做结构化提取。

【门诊病历·肝胆胰外科】性别男，高龄患者，初诊。主诉：发热5天。现病史：20号左右出现发热持续约5天，伴轻微腹痛。既往史：否认糖尿病，冠心病行支架治疗15年余口服阿司匹林，既往肝血管瘤病史。体格检查：上腹部轻压痛。外院核磁：肝右叶占位合并出血，胆囊结石。诊断：肝占位性病变（血管瘤？）；胆囊结石。

【MRI报告】肝MRI平扫+增强+MRCP+DWI。所见：肝内见团块状T1WI混杂信号T2WI混杂信号，大小约59.7mm×63.1mm×63.4mm，DWI不均匀稍高信号，增强后动脉期不均匀明显强化，延迟期强化减退，可见假包膜，病灶与门静脉左支分支关系密切。胆囊见小结节状充盈缺损。左肾见类圆形T2WI高信号。印象：1.肝脏占位，考虑原发性肝细胞癌；2.胆囊小结石；3.左肾囊肿。

【病理报告】（肝脏穿刺组织）结合组织学图像及免疫标记结果，考虑为高分化肝细胞性肝癌。送检组织极少。免疫组化：Glypican-3（-），Hepatocyte（+），CD34（血管+），AFP（-），CK7（-），CK19（-），Ki-67（8%+），GS（+），β-catenin（+），HSP70（+），INSM1（-）。
```

✅ **预期要点**：主要诊断=肝细胞癌(C22.0/8170/3)；良恶性=恶性(病理依据)；影像识别快进快出+假包膜；免疫组化逐项解析；门诊"血管瘤?"→病理"HCC"记为诊断演变warning；覆盖度小结显示多份文书均覆盖。

## 用例2（患者B）——晚期HCC伴门脉癌栓【考点：分期映射+乙肝肝硬化背景】

**这条展示**：门脉癌栓→CNLC Ⅲa/BCLC C 分期推导，乙肝五项+HBV-DNA结构化，肝硬化背景关联。

📋 **输入文本**：
```
请对以下肝病病历做结构化提取。

【MRI报告】肝脏MRI平扫+增强+MRCP+DWI（细胞外钆剂+普美显）。所见：肝实质内多发结节状团块状T1WI稍低T2WI稍高信号，DWI高信号，较大者约60.2mm×61.2mm×71.4mm，动态增强动脉期明显不均匀强化，门脉期及延迟期强化廓清，肝胆期低信号。肝中静脉、门静脉右支及主干内斑片状低信号充盈缺损。脾大。食管下段胃底静脉曲张。腹腔内少量积液。印象：1.肝脏多发占位，考虑肝癌伴肝内多发转移；2.肝中静脉、门静脉右支癌栓形成，门脉主干栓子形成；3.肝硬化，脾大，食管下段胃底静脉曲张，少量腹水；4.双肾囊肿。

【入院记录】男，老年。主诉：体检发现肝占位4日余。现病史：外院超声示肝多发实性占位，门静脉增宽伴栓子，MR考虑HCC。既往：乙型肝炎多年未治疗，血小板减少多年。个人史：无吸烟史，无饮酒史。

【检验】HBV-DNA：1.20E+4（异常H）。乙肝：HBsAg 221.860(A)，Anti-HBe 0.010(A)，Anti-HBc 8.160(A)。肿瘤三项+AFP-L3+PIVKA-II：异常报告。

【病案首页】主要诊断：肝细胞癌。其他诊断：乙型肝炎小三阳；脾大；单纯性肾囊肿；少量腹腔积液。病理诊断：肝细胞癌。
```

✅ **预期要点**：主诊断=肝细胞癌(C22.0)；门脉癌栓+多发→CNLC Ⅲa / BCLC C；乙肝小三阳+HBV-DNA结构化；肝硬化(脾大/静脉曲张/腹水为依据)；个人史"无吸烟无饮酒"如实记录(非"未提及")。

## 用例3（患者C）——小肝癌消融术后【考点：治疗信息提取+诊断未分型映射】

**这条展示**：微波消融术细节提取，病案首页"肝恶性肿瘤"未分型→C22.9映射，抗病毒治疗，肿瘤标志物含范围符号(<1.2)。

📋 **输入文本**：
```
请对以下肝病病历做结构化提取。

【门诊病历·肝病科】男，中年，初诊。主诉：乙肝多年，MRI发现肝癌。现病史：乙肝多年，因颅脑出血行手术治疗。住院期间MRI提示肝脏右叶异常信号，考虑小肝癌可能。诊断：肝占位性病变。

【首次病程】确诊乙肝20余年，间断停用抗病毒药10余年，现口服艾米替诺福韦抗病毒。外院肝MR（肝特异性对比剂）印象：肝右叶异常信号考虑小肝Ca可能；肝硬化并多发再生结节；双肾囊肿。

【术后病程】今日局麻下行超声引导下肝病损微波消融术，人工腹水1000ml。超声造影：肝右叶包膜下一大小约17×16mm动脉期强回声灶，考虑肝Ca。15G消融针移动消融法，对1处病灶行消融。术后彩超未见明显出血。

【病案首页】主要诊断：肝恶性肿瘤（TA期）。其他诊断：恶性肿瘤介入治疗；乙型肝炎肝硬化；继发性血小板减少；脑出血恢复期。

【检验】AFP-R：55.95 ng/mL(H，参考0-8.1)；CEA：1.14(参考0-5)；CA19-9：<1.2(参考0-37)。
```

✅ **预期要点**：治疗=超声引导微波消融术(单病灶/人工腹水细节)；抗病毒=艾米替诺福韦；主诊断"肝恶性肿瘤"未分型→C22.9(标注可结合影像为C22.0)；单发1.7cm→CNLC Ⅰa；AFP↑结构化；CA19-9的"<1.2"原文照录不强转。

## 用例4（患者D）——FNH良性【考点：良恶性鉴别+避免误编 ★鲁棒性王牌】

**这条最关键**：影像"考虑FNH，恶性不除外"措辞模糊，但病理确诊良性FNH → 必须判良性、编D13.4、ICD-O-3标"不适用"，不被影像疑似措辞带偏。

📋 **输入文本**：
```
请对以下肝病病历做结构化提取。

【门诊病历·肝胆胰外科】男，老年。主诉：发现肝占位性病变。现病史：外院普通磁共振发现肝占位，肝病科建议增强MR。既往乙肝病毒量高服药治疗。否认饮酒史。

【MRI报告】肝脏平扫+增强+DWI。所见：肝左叶萎缩。肝右叶见类圆形T1WI稍低T2WI高信号，边界清晰，DWI稍高信号，约18.9mm×13.9mm，增强动脉期明显强化，延迟期稍减退；肝脏另见多发类圆形T2WI高信号无强化灶。印象：1.肝右叶占位，考虑FNH可能，恶性肿瘤性病变不能完全除外，建议结合实验室检查；2.胆囊结石；胆囊腺肌症；3.肝脏多发囊肿；脾囊肿；4.右肾囊肿。

【病理报告】（肝部分切除标本）肝组织中度水肿变性、轻度脂肪变性，汇管区纤维组织增生伴炎症细胞浸润。肿块镜下：纤维组织将肝组织分隔呈结节状，纤维间隔内见厚壁大血管、小胆管增生及淋巴细胞浸润，肝细胞形态正常，肝小叶结构消失，肝板厚度不超过2层。结合形态学和免疫组化，支持为：肝局灶性结节状增生（FNH）。免疫组化：GS（地图状+），CD34（部分区域毛细血管化），Glypican-3（-）。特殊染色：网状纤维染色（未见明显断裂）。

【病案首页】主要诊断：肝结节性局灶性增生。其他诊断：胆囊结石伴慢性胆囊炎；胆囊腺肌症；肝囊肿；肺气肿。
```

✅ **预期要点**：良恶性=**良性**(以病理为准)；ICD-10=**D13.4**；ICD-O-3=**不适用(FNH瘤样病变)**；GS必须保留"**地图状**+"修饰(对应FNH而非HCC)；网状纤维"无断裂"+CD34毛细血管化作为良性依据；影像"恶性不除外"记为疑似措辞但被病理推翻，写入warnings；不分期(良性)。

## 用例5——非规范文本鲁棒性【考点：缩写/口语/信息散落/脱敏】

**这条展示**：极度不规范的真实门诊速记——大量缩写、口语、信息倒装、脱敏占位，考验鲁棒提取与"零虚构"。

📋 **输入文本**：
```
请对以下肝病病历做结构化提取。

男，老年。乙肝20+y，小三阳，TMF抗病毒中。外院MR见肝右叶占位3a期，门脉癌栓，HCC可能大。AFP偏高。LSM/CAP 8.8/284提示肝硬化。既往史已脱敏。体格检查：人未至。拟收住肝胆外科。家族史否认肿瘤。具体病理待入院后穿刺。
```

✅ **预期要点**：缩写展开(HCC→肝细胞癌、TMF→替诺福韦类、3a期→CNLC Ⅲa、y→年)；门脉癌栓→CNLC Ⅲa/BCLC C；小三阳结构化；LSM/CAP→肝硬化依据；"人未至"→体征未提及；"病理待入院"→病理字段标未提及+提示"未含病理报告"；良恶性=疑似恶性(无病理，仅影像"可能大")**不强判恶性**；AFP"偏高"无数值→记定性升高但标注无具体值；家族史"否认肿瘤"如实记录。展示在信息稀疏+不规范下仍不虚构。
