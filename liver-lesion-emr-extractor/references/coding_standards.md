# 标准化编码映射规则（coding_standards.md）

本文件定义肝占位/肝癌结构化提取的国际/国家标准编码映射，是赛道2"标准化映射合规性与一致性"考核的核心依据。

> **映射原则**：以**病理金标准 > 临床确诊 > 影像疑似**为编码可信度排序。疑似/未定性病灶用未特指码并标注低置信度。每个映射输出 `match_level`（P0/P1/P2）与 `confidence`。

---

## 一、ICD-10 诊断编码对照（肝脏病变）

| 诊断 | ICD-10 | 适用条件 | 备注 |
|------|--------|---------|------|
| 肝细胞癌 | **C22.0** | 病理或临床确诊HCC | 首选 |
| 肝内胆管癌 | C22.1 | 病理确诊ICC | |
| 肝恶性肿瘤，未特指 | **C22.9** | 确诊恶性但分型未明（如病案首页"肝恶性肿瘤"） | 患者C适用 |
| 肝继发性恶性肿瘤（转移） | C78.7 | 明确为转移性 | |
| 肝良性肿瘤（含FNH、肝细胞腺瘤） | **D13.4** | 病理确诊良性 | 患者D（FNH）适用 |

**FNH 说明**：肝局灶性结节性增生（FNH）属**瘤样病变/反应性增生**，并非真性肿瘤，**ICD-O-3 不编形态学码**。映射时 `icdo` 填 `不适用（FNH为瘤样病变，非肿瘤性）`，避免误编。这是一致性合规要点。

## 三、肝癌分期映射

### 3.1 中国 CNLC 分期（《原发性肝癌诊疗指南》）

| CNLC 分期 | 核心条件（简化判定） |
|-----------|---------------------|
| **Ⅰa** | 单个肿瘤≤5cm，无血管侵犯/肝外转移，PS 0-2，肝功Child-Pugh A/B |
| **Ⅰb** | 单个>5cm，或2-3个≤3cm，无血管侵犯/肝外转移 |
| **Ⅱa** | 2-3个肿瘤、最大>3cm，无血管侵犯/肝外转移 |
| **Ⅱb** | ≥4个肿瘤，无血管侵犯/肝外转移 |
| **Ⅲa** | 有血管侵犯（**门脉/肝静脉癌栓**） | 
| **Ⅲb** | 有肝外转移（淋巴结/远处） |
| **Ⅳ** | Child-Pugh C / PS 3-4 |

> 判定所需信息不足时填 `信息不足无法分期`，**不强行套档**。

### 3.2 BCLC 分期

| BCLC | 条件 |
|------|------|
| 0（极早期） | 单个≤2cm，Child A，PS 0 |
| A（早期） | 单个或≤3个且≤3cm，Child A-B，PS 0 |
| B（中期） | 多结节，Child A-B，PS 0 |
| C（进展期） | 门脉侵犯/肝外转移，PS 1-2 |
| D（终末期） | Child C 或 PS 3-4 |

### 3.3 AJCC 第8版 TNM（如原文提供则校验）

| 要素 | 简化规则 |
|------|---------|
| T1a | 单个≤2cm，无血管侵犯 |
| T1b | 单个>2cm，无血管侵犯 |
| T2 | 单个>2cm伴血管侵犯，或多个≤5cm |
| T3 | 多个，至少一个>5cm |
| T4 | 累及门脉/肝静脉主要分支，或侵犯邻近器官 |
| N1 | 区域淋巴结转移 |
| M1 | 远处转移 |

**自洽校验**：M1 → 必为Ⅳ期/BCLC C/CNLC Ⅲb；门脉癌栓 → CNLC Ⅲa/BCLC C。冲突写入 warnings。

## 四、真实病例映射示例（来自本项目4份病例）

| 患者 | 主要诊断（原文） | 良恶性 | ICD-10 | ICD-O-3 | 分期提示 |
|------|----------------|--------|--------|---------|---------|
| A | 高分化肝细胞性肝癌（穿刺病理） | 恶性 | C22.0 | 8170/3 | 单发约6cm，无明确转移→CNLC Ⅰb（信息有限，标注置信度） |
| B | 肝细胞癌伴肝内多发转移、门脉癌栓 | 恶性 | C22.0 | 8170/3 | 门脉癌栓+多发→CNLC Ⅲa / BCLC C |
| C | 肝恶性肿瘤（小肝癌，消融术后） | 恶性 | C22.9（首页未分型）→可C22.0 | 8170/3（临床） | 单发约1.7cm→CNLC Ⅰa / BCLC A |
| D | 肝局灶性结节性增生（FNH，病理确诊） | **良性** | **D13.4** | 不适用 | 不分期（良性） |

> 患者D是**良恶性鉴别**与**避免误编**的关键示范：影像曾"考虑FNH可能，恶性不除外"，但病理确诊良性 → 必须以病理为准编 D13.4，不得受影像疑似措辞影响编恶性码。

## 五、匹配优先级与置信度

| 优先级 | 条件 | 置信度 |
|:---:|------|:---:|
| P0 | 病理确诊 + 标准名完全对应 | 0.95+ |
| P1 | 临床确诊/影像典型 + 上位匹配 | 0.85-0.94 |
| P2 | 影像疑似/未定性（"考虑××""不除外"） | 0.60-0.84 |
| P3 | 信息不足，仅占位无定性 | <0.60，标注"待人工复核" |
