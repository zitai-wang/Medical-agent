# 字段定义体系（field_schema.md）

本文件定义肝占位/肝癌病历结构化提取的 **8 大类、共 40+ 个字段**。每个字段含：字段名、说明、取值规范、常见来源文书、缺失时填写规则。

> **通用规则**：所有字段缺失时填 `未提及`；每个有值字段须附 `source_text`（原文片段）与 `source_doc`（来源文书类型）。

---

## 一、患者基本信息 patient_info

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 年龄段 | `age` | 脱敏病历常为"中年/老年/高龄患者"，如实记录；如有具体年龄则记数值 | 各文书抬头 |
| 性别 | `sex` | 男/女 | 各文书抬头 |
| 体格参数 | `anthropometry` | 身高/体重/BMI，如 "162cm/55kg"；无则未提及 | 入院记录、体格检查 |
| 就诊科室 | `department` | 如 肝胆胰外科、感染病科、肝病科；保留原文 | 门诊病历、病案首页 |
| 就诊状态 | `visit_status` | 初诊/复诊/住院 | 门诊病历 |
| 入院途径 | `admission_route` | 门诊/急诊/转院 | 病案首页 |

## 二、主诉与现病史 clinical_presentation

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 主诉 | `chief_complaint` | 原文照录，如"体检发现肝占位4日余" | 门诊病历、入院记录 |
| 现病史概要 | `hpi_summary` | 归纳起病-检查-就诊经过，保留关键时间节点与外院结果 | 现病史、首次病程 |
| 症状 | `symptoms` | 列举：发热/腹痛/腹胀/黄疸/消瘦/乏力等；区分"有"与"否认" | 现病史 |
| 体征 | `physical_signs` | 腹部压痛、肝脾触诊、Murphy征、移动性浊音等 | 体格检查 |

## 三、高危背景与既往史 risk_background

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 乙肝状态 | `hbv` | 有/无；病程（如"乙肝20余年"）；大三阳/小三阳；抗病毒药（ETV/TMF/TDF/艾米替诺福韦） | 既往史、现病史 |
| HBV-DNA | `hbv_dna` | 数值+单位+异常标识，如 "1.20E+4 升高" 或 "<100" | 检验摘录 |
| 乙肝五项 | `hbv_panel` | HBsAg/HBsAb/HBeAg/HBeAb/HBcAb 定量结果与定性（大三阳/小三阳） | 检验摘录 |
| 肝硬化 | `cirrhosis` | 有/无；依据（影像"肝硬化"、LSM值、脾大、食管胃底静脉曲张、腹水） | 影像、现病史 |
| 肝纤维化指标 | `fibrosis` | LSM/CAP 值，如 "8.8/284" | 检验、无创诊断 |
| 其他既往史 | `past_history` | 冠心病/糖尿病/高血压/手术史等；区分"有"与"否认" | 既往史 |
| 个人史 | `personal_history` | 吸烟/饮酒/疫水接触等；如实记录"无吸烟史"等明确否认 | 个人史 |
| 家族史 | `family_history` | 肿瘤/遗传病家族史；明确否认也记录 | 家族史 |

## 四、影像学 imaging

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 检查方式 | `imaging_modality` | MRI平扫+增强+MRCP+DWI / CT平扫 / 超声造影；保留对比剂（普美显/钆剂/SonoVue） | 影像报告 |
| 病灶部位 | `lesion_site` | 肝右叶/肝左叶/具体肝段；侧别 | 影像报告 |
| 病灶大小 | `lesion_size` | 三径或最大径，原文数值，如 "59.7mm×63.1mm×63.4mm" | 影像报告 |
| 病灶数目 | `lesion_number` | 单发/多发 | 影像报告 |
| 信号/密度 | `signal` | T1WI/T2WI/DWI 信号描述 | 影像报告 |
| 强化方式 | `enhancement` | 动脉期强化、门脉/延迟期廓清（"快进快出"）；或地图状强化（FNH）；或无强化（囊肿） | 影像报告 |
| 特征征象 | `imaging_signs` | 假包膜、马赛克征、瘢痕、再生结节等 | 影像报告 |
| 血管受累 | `vascular` | 门脉/肝静脉癌栓、栓子；血管关系 | 影像报告 |
| 影像印象 | `imaging_impression` | 影像诊断印象原文（含"考虑××""不除外"等定性程度） | 影像报告印象 |
| 伴随发现 | `incidental` | 胆囊结石/腺肌症、肾囊肿、脾大、腹水等 | 影像报告 |

## 五、病理与免疫组化 pathology

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 病理诊断 | `pathology_dx` | 病理报告结论原文，如"高分化肝细胞性肝癌""支持为FNH" | 病理报告 |
| 标本类型 | `specimen` | 穿刺组织/部分切除标本/胆囊切除标本 | 病理报告 |
| 分化程度 | `grade` | 高/中/低分化；Edmondson分级（如有） | 病理报告 |
| 免疫组化 | `ihc` | 逐项标记+结果，原文照录，如 "Glypican-3(-), Hepatocyte(+), AFP(-), Ki-67(8%+), GS(+), β-catenin(+), HSP70(+)" | 病理报告 |
| 特殊染色 | `special_stain` | 网状纤维染色等 | 病理报告 |
| 病理备注 | `path_note` | "送检组织极少""经多位主任阅片"等限定说明 | 病理报告 |

## 六、肿瘤标志物 tumor_markers

每个标志物为一条记录：`{name, value, unit, ref_range, abnormal}`

| 标志物 | 说明 | 临床意义提示（仅记录不判读） |
|--------|------|------|
| AFP（甲胎蛋白） | ng/mL | HCC 主要标志物 |
| AFP-L3 | % | HCC 特异性更高 |
| PIVKA-II / 异常凝血酶原 | mAU/mL | HCC 标志物 |
| CEA（癌胚抗原） | ng/mL | 消化道肿瘤 |
| CA19-9 | U/mL | 胆道/胰腺 |
| CA125 | U/mL | — |

> 取值须含：数值、单位、参考范围、异常标识（升高/降低 或 H/L）。如 `AFP 55.95 ng/mL 升高 (参考 0-8.1)`。

## 七、诊断与分期 diagnosis_staging

| 字段 | 字段名 | 取值规范 | 来源文书 |
|------|--------|---------|---------|
| 门诊诊断 | `outpatient_dx` | 原文 | 门诊病历、病案首页 |
| 入院诊断 | `admission_dx` | 原文 | 入院记录、病案首页 |
| 主要诊断 | `main_diagnosis` | 病案首页"主要诊断" | 病案首页 |
| 其他诊断 | `other_diagnoses` | 列举合并症/伴随诊断 | 病案首页 |
| ICD-10 | `icd10` | 见 coding_standards.md | 映射生成 |
| 良恶性判定 | `benign_malignant` | 恶性/良性/疑似待定（依据病理金标准优先） | 综合判定 |
| 判定依据 | `bm_basis` | 列出支持依据（影像/病理/免疫组化/标志物） | 综合判定 |
| CNLC分期 | `cnlc` | Ⅰa/Ⅰb/Ⅱa/Ⅱb/Ⅲa/Ⅲb/Ⅳ 或 "信息不足无法分期" | 映射推导 |
| BCLC分期 | `bclc` | 0/A/B/C/D 或 "信息不足" | 映射推导 |
| TNM | `tnm` | 如原文给出按AJCC第8版校验，否则"未提供" | 映射校验 |

## 八、治疗信息 treatment

每条治疗为一条记录：`{type, detail, date, source_text}`

| 治疗类型 | 字段值示例 | 来源文书 |
|---------|-----------|---------|
| 手术切除 | 肝部分切除术/肝段切除 | 病程、病案首页 |
| 消融 | 超声引导下微波消融术（含人工腹水、消融针、病灶数） | 术后病程 |
| 介入 | TACE/恶性肿瘤介入治疗 | 病案首页、病程 |
| 系统治疗 | 靶向/免疫/化疗（如有） | 病程、医嘱 |
| 抗病毒 | ETV/TMF/TDF/艾米替诺福韦 | 既往史、医嘱 |
| 治疗计划 | 拟行检查/手术计划 | 治疗计划、随访医嘱 |

## 字段覆盖度统计口径

输出"质量小结"时：
- `total` = 本病历**理论应覆盖**的字段数（按文书类型动态：仅门诊病历则病理类字段不计入分母，标注 "本次输入不含病理报告"）
- `filled` = 实际填出值（非"未提及"）的字段数
- `missing` = 标注"未提及"的字段数
- 覆盖度 = filled / total，用于体现赛道2"覆盖度"考核
